# Вилочный погрузчик — симуляция (Unity тестовое задание)

## Описание проекта

Данный проект представляет собой реализацию тестового задания на позицию Unity-программиста. Реализована базовая симуляция вилочного погрузчика с управлением с клавиатуры, физикой перемещения, подъёмом/опусканием вилы, системой топлива и минимальной визуализацией.

**Важно отметить**, что проект выполнен в рамках тестового задания и не является законченным продакшен-решением. В нём сознательно не учтён ряд аспектов, таких как:

* Обработка ситуации с закончившимся топливом;
* Управление камерой и возможность выхода из автомобиля;
* Некоторые компоненты реализованы в упрощённом виде для скорости разработки.

Например, управление игровым циклом выполнено простейшим образом, тогда как в реальном проекте стоило бы реализовать полноценную стейт-машину.

---

## Реализованные функции

### Основные:
* Старт сцены с затемнением экрана (Fade-in из чёрного);
* Управление погрузчиком с клавиатуры:
  * Запуск двигателя — `T`;
  * Движение — `WASD` (на основе Wheel collider);
  * Подъём/опускание вилы — `Q`/`E`;
* Груз в виде куба на подставке (имитация поддона);
* Появление груза на стартовой площадке с анимацией спуска и вращения (5 секунд);
* Система топлива:
  * Уровень топлива снижается при работе машины;
  * При уровне ниже 50% скорость движения уменьшается вдвое;
  * Отображение уровня топлива в кабине (в виде шкалы);
* Цель — переместить груз со стартовой площадки на площадку разгрузки (отмечена серым)

### Дополнительные (опциональные):
* При размещении груза на площадке разгрузки он поднимается вверх с ускорением и вращением по всем осям;
* Камера заднего вида выводится на монитор приборной панели в кабине;
* Простая приборная панель с отображением уровня топлива.

## Структура проекта (Assembly Definitions)

Код разделён на логические сборки:

* **Core** — интерфейсы базовой логики
* **App** — реализации базовой логики
* **Compilation** — для связывания компонентов между сборками
* **Input** — обработка ввода;
* **Visual** — всё, что связано с визуалом: анимации, UI, эффекты.
